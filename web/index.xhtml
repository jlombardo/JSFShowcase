Welcome to WCTC's JavaServer Faces Demo Project
(Updated Dec. 19, 2011)

This Netbeans project provides a variety of demonstrations of JSF v2 and facelets programming techniques, using both JSF Managed Beans and CDI implementations, plus Spring Framework core and security services.

Supporting HTML Comments

In the source code for the XHTML pages you will find HTML comments. Normally, these comments are not allowed near some JSF/Facelets tags. However, this can be allowed by configuring your web app to allow them. Add this to your web.xml file:

            <context-param>
                <param-name>javax.faces.FACELETS_SKIP_COMMENTS</param-name>
                <param-value>true</param-value>
            </context-param>

PrimeFaces v2.2.1 JSF Components

Many JSF components used in this version are from PrimeFaces v2.2.1. Other 3rd party libraries needed are the Spring core and security .jar files, plus the the JDBC driver for MS-SQL Server.

This project has been tested only on Glassfish 3.1.1. which contains Mojarra 2.1.0.

Also, beginning with this demo i18n properties files (see "default package") are used for some of the text in the view layer (mainly the title, but the minValue and maxValue as well). "i18n" is the term used for internationalizing an application. This is done by providing a common base name for the file, plus a language extension, such as "en" for English or "de" for German.
